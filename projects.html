Digital System Design

Completed

−

Project Overview

This project involved designing a digital tram control system for an airport transportation setting using a finite state machine (FSM). The client requirement was to create a control system based on four given signals, while the overall FSM structure and implementation strategy were left open-ended. I designed a 6-state tram control system that determines motion and stopping behavior based on the relative progression of states rather than absolute floor position, making the design applicable regardless of the number of stops.

A key objective of my design was hardware minimization. Starting from the state diagram and present-state / next-state relationships, I repeatedly refined the state table and applied Karnaugh map (K-map) minimization to exploit symmetry and repeated logic patterns. By maximizing useful groupings and reducing Boolean expressions, I implemented the system with only 3 D flip-flops, 5 AND gates, 3 OR gates, and 2 NOT gates. The project was verified through digital simulation and received a full score.

My Experience / Project Concept

This project helped me strengthen my understanding of sequential logic design beyond basic FSM construction. I learned how to move from a conceptual control problem to a complete hardware-oriented implementation by building a state diagram, deriving a state table, simplifying logic with K-maps, and validating behavior through simulation. More importantly, I focused on designing the control logic as efficiently as possible rather than only making it function, which gave me stronger insight into digital hardware optimization.
